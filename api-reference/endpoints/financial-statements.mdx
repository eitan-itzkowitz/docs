Retrieve Financial Statements
GET /financial-statements
Fetch structured financial statements, including balance sheets, income statements, and cash activity reports.

Request Parameters
Pass the following query parameters in the request.

Query Parameters
type | string | Required
The type of financial statement. Possible values: balance_sheet, income_statement, cash_activity_report.

basis | string | Required
The accounting basis for the report. Options: cash, accrual.

start_date | string | Required
The start date for the financial statement in YYYY-MM-DD format.

end_date | string | Required
The end date for the financial statement in YYYY-MM-DD format.

interval | string | Required
The reporting frequency. Options: monthly, quarterly, yearly.

Response Structure
Returns a JSON object representing the requested financial statement.

Financial Statement Object
type | string | Required
The type of financial statement returned. Possible values: balance_sheet, income_statement, cash_activity_report.

basis | string | Required
The accounting basis of the statement: cash or accrual.

currency | string | Required
The currency code in ISO 4217 format (e.g., USD).

key | string | Required
A unique identifier for the statement type.

name | string | Required
The human-readable name of the financial statement (e.g., "Income Statement").

intervals | array | Required
The reporting periods included in the response.

Interval Object (Nested in intervals)
key | string | Required
A unique key identifying the reporting interval.

periods | array | Required
A list of periods covered by this interval (e.g., ["2024-01", "2024-02"]).

Line Item Object (Nested in lines)
id | string | Required
Unique identifier for the statement line.

parentId | string \| null | Optional
The ID of the parent line if this is a subcategory. null for top-level items.

childIds | array | Optional
An array of child line IDs if this line is a parent category.

depth | integer | Required
The hierarchy level of the line within the financial statement.

name | string | Required
The label for this line (e.g., "Total Revenue").

type | string | Required
Indicates whether this line represents a summary (rollup), a grouping of accounts (parent_account_group), or an individual account.

amountType | string | Required
Specifies whether this line is a credit or debit.

intervals | array | Required
Financial values for each period in the response.

Monetary Value Object (Nested in intervals)
monetary_value.amount | string | Required
The financial amount for this line, returned as a string for precision.

monetary_value.currency | string | Required
The currency in which the amount is reported.

Example Request

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
