## Monetary Value Model

The **MonetaryValue** object provides a standardized structure for handling financial amounts.

:::info
This model ensures **all monetary values are correctly formatted** in terms of **amount and currency**.
:::

### **Schema**
```json
{
  "amount": 1234.56, // The monetary amount with up to two decimal places
  "currency": "USD" // ISO 4217 currency code (e.g., USD, EUR)
}
Important Notes:

amount supports both positive and negative values (negative = refunds/credits).
currency follows the ISO 4217 standard for international compatibility.
