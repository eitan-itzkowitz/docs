## Tax Rate Model

The **TaxRate** object stores tax configuration details, including jurisdiction, calculation type, and effective dates.

:::info
This model supports **various tax types**, including **sales tax, income tax, and payroll withholding**.
:::

### **Schema**
```json
{
  "id": "taxrate-sales-CA", // Unique tax rate identifier
  "createdAt": "2023-01-01T00:00:00Z", // Timestamp when the tax rate was created
  "updatedAt": "2024-01-15T12:00:00Z", // Timestamp of last update
  "company": {
    "id": "company-987abc123def",
    "legalName": "Bright Smiles Orthodontics"
  },
  "displayName": "California Sales Tax", // Readable name for the tax rate
  "taxType": "sales_tax", // Type of tax (e.g., sales tax, income tax)
  "jurisdictionType": "State", // Federal, State, City, etc.
  "jurisdictionName": "California", // Name of the jurisdiction
  "calculationType": "percentage_flat", // Defines how the tax is calculated
  "calculation_parameters": {
    "percentageRate": 7.5 // Flat percentage tax rate
  },
  "effectiveDate": "2023-01-01T00:00:00Z", // Date when tax rate becomes active
  "expirationDate": null // If null, the tax rate is still active
}
