# Get Started with Puzzle

The Puzzle API enables seamless, automated accounting workflows. Here’s how to get up and running.

##  Start in 3 Steps

<AccordionGroup>

<Accordion icon="edit" title="1. Request API Access">
  Submit a short form to gain credentials for testing and development.  
  <Button href="https://docs.google.com/forms/d/e/1FAIpQLScCeVQ9E78o6afmUzWDcDioIeMNJ3ljs9oyhxYc-E2Mj2C5Yw/viewform" variant="primary">Request Access</Button>
</Accordion>

<Accordion icon="play-circle" title="2. Explore the Sandbox">
  Test our API in a live sandbox environment—no setup required.  
  <Button href="https://puzzle.io/sandbox" variant="secondary">Open Sandbox</Button>
</Accordion>

<Accordion icon="key" title="3. Get Your API Credentials">
  Our team reviews access requests within **1–2 business days**. Once approved, you'll receive sandbox API keys to start building.
</Accordion>

</AccordionGroup>

---

## Next Steps

1. **Receive your API sandbox credentials** via email.  
2. **Start integrating** with our [API Documentation](/api-reference/introduction).  
3. **Test in the sandbox** before moving to production.  

For full production access, additional verification may be required.
